# TMinder Backend

TMinder (TV-Minder) is an enterprise-grade backend service designed to help users track their favorite TV series, manage watchlists, and stay updated on episode release schedules.

## 🌟 Project Overview
The goal of TMinder is to provide a robust, scalable API that aggregates series data and provides personalized tracking for users. It is built with a focus on maintainability and clean code.

---

## 🏗 Architecture & Tech Stack
We follow **Clean Architecture** to ensure the business logic remains independent of external frameworks.

- **Framework**: Spring Boot 3.4.2
- **Language**: Java 21
- **Build Tool**: Gradle (Kotlin DSL)
- **Layers**: Domain, Application, Infrastructure, and API.
